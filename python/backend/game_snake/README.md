## the_snake

## Стек технологий
- Python - основной язык разработки проекта  
- Pygame - библиотека для создания 2D-игры, работы с графикой, игровым окном, событиями клавиатуры и игровым циклом (game loop). Используется для отрисовки объектов, управления FPS и обновления экрана  
- random (choice) - используется для генерации случайных координат появления яблока на игровом поле без пересечений со змейкой  
- ООП (объектно-ориентированное программирование) - архитектура проекта построена на классах и наследовании:
  - базовый класс GameObject для общих свойств объектов
  - классы Snake и Apple как наследники с переопределением поведения  
  - инкапсуляция логики движения и отрисовки внутри объектов  
- Игровая логика и математика координат - работа с сеткой (grid-based movement), использование координатной системы, циклическое поле через modulo (%), обработка столкновений и роста змейки  
- Обработка событий Pygame - управление через клавиатуру (KEYDOWN), обработка выхода из игры, ограничение разворота змейки через словарь направлений TURNS
