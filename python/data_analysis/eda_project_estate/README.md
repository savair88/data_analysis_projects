## Стек
- Python (обработка данных)
- Pandas, NumPy (анализ данных)
- Matplotlib, Seaborn (визуализация)
- Jupyter Notebook (исследование данных)

## Основные методы и подходы
- Предобработка данных:
  - обработка пропусков (fillna, удаление, логические замены)
  - приведение типов данных (to_datetime, astype)
  - работа с аномалиями и выбросами

- Feature Engineering:
  - расчет новых признаков (цена за м², день/месяц/год публикации)
  - категоризация данных (тип этажа)
  - преобразование временных данных

- Исследовательский анализ (EDA):
  - анализ распределений (hist, boxplot)
  - выявление выбросов
  - расчет статистик (mean, median)

- Анализ зависимостей:
  - scatter plots
  - группировки (groupby)
  - агрегаты (median, mean)
  - выявление факторов, влияющих на цену

- Работа с геоданными:
  - анализ расстояния до центра
  - сравнение по локациям (top городов)

- Визуализация данных:
  - гистограммы
  - столбчатые диаграммы
  - scatter plots
