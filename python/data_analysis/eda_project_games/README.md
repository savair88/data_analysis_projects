## Стек
- Python — основной язык анализа данных
- Pandas — обработка и трансформация табличных данных
- NumPy — работа с числовыми операциями
- Matplotlib — базовая визуализация данных
- SciPy (scipy.stats) — статистические тесты и проверка гипотез

## Основные навыки и функции
- Очистка и подготовка данных:
  - приведение типов (`astype, to_numeric`)
  - обработка пропусков (replace, isna)
  - нормализация названий колонок
  - удаление дубликатов (duplicated, drop)
- Feature engineering:
  - расчет суммарных продаж (total_sales)
  - объединение данных по регионам
  - создание аналитических признаков
- Исследовательский анализ (EDA):
  - анализ распределений (hist, boxplot)
  - исследование динамики во времени
  - анализ жизненного цикла платформ
- Агрегация и сводные таблицы:
  - groupby, agg
  - pivot_table
  - расчет долей и нормализация данных
- Анализ зависимостей:
  - корреляция (corr)
  - визуализация scatter plot
  - сравнение метрик по категориям
- Сегментация пользователей:
  - анализ по регионам (EU, NA, JP)
  - выявление предпочтений по платформам и жанрам
- Проверка статистических гипотез:
  - t-test (scipy.stats.ttest_ind)
  - формулировка H0 / H1
  - интерпретация p-value
