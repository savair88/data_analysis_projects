# Стек  
Python — основной язык анализа данных  
Pandas — обработка и трансформация табличных данных  
NumPy — работа с числовыми операциями  
Matplotlib — базовая визуализация данных  
Seaborn — статистическая визуализация данных  
Scikit-learn — машинное обучение и пайплайны  
CatBoost — градиентный бустинг для табличных данных  
Phik — анализ корреляций смешанных типов признаков  

---

# Основные навыки и функции  

## Очистка и подготовка данных:
- приведение типов (`astype`, `to_datetime`)
- обработка пропусков (`fillna`, замена значений на категорию `no_info`)
- переименование и унификация колонок (`rename`, `columns`)
- обработка дат и создание признаков времени
- удаление дубликатов (`duplicated`, `drop`)

---

## Feature engineering:
- создание признака жизненного цикла клиента (`duration`)
- создание целевой переменной (`target`)
- объединение нескольких источников данных (`join` по `customer_id`)
- удаление мультиколлинеарных признаков
- преобразование бинарных признаков в единый формат

---

## Исследовательский анализ (EDA):
- анализ распределений (`histplot`, `boxplot`)
- анализ категориальных признаков
- исследование оттока клиентов по сегментам
- визуализация взаимосвязей (`pairplot`)
- анализ аномалий и выбросов

---

## Агрегация и анализ данных:
- `groupby`, `agg` — расчет статистик
- `value_counts` — анализ категорий
- расчет долей оттока клиентов
- анализ длительности контрактов

---

## Анализ зависимостей:
- корреляционный анализ (`phik_matrix`)
- анализ взаимосвязей признаков
- выявление мультиколлинеарности
- визуализация heatmap

---

## Машинное обучение:
- построение ML-пайплайнов (`Pipeline`)
- предобработка данных (`ColumnTransformer`)
- кодирование категориальных признаков (`OneHotEncoder`)
- масштабирование (`StandardScaler`)
- обучение моделей:
  - Logistic Regression
  - CatBoostClassifier

---

## Подбор моделей и оценка:
- `GridSearchCV` — подбор гиперпараметров
- ROC-AUC как основная метрика качества
- ROC-кривая (`roc_curve`)
- матрица ошибок (`confusion_matrix`)
- анализ feature importance (CatBoost)
